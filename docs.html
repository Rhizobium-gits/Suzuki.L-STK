アプリケーション概要

Suzuki.L-STKは鈴木治夫研究室のサンプリング作業管理ツールです。複数の作業者のタスク進行状況を一元管理し、効率的な実験進行をサポートします。

使用方法

  1. 「Let's start Sampling!!」ボタンをクリックして開始
  2. 作業者・作業登録フォームに情報を入力
    • 作業者名：担当者の名前を入力
    • 作業名：実行するタスクの名前
    • 予定作業時間：分単位で入力
  3. 「追加」ボタンで作業者とタスクを追加（複数追加可能）
  4. 登録が完了したら「登録完了」ボタンをクリック
  5. 作業者カードの「作業開始」ボタンでタスク開始
  6. 作業中のステータス操作
    • 「作業中」ボタンクリックでメニュー表示
    • 「作業を終わる」で次のタスクへ移動
    • 「一時停止」でタスクを一時停止
    • 「新規作業追加」で追加タスク登録
  7. すべての作業完了後、「サンプリング終了」ボタンでリセット

機能説明

  • リアルタイムクロック表示：現在時刻を表示
  • プログレスバー：タスク進行度を視覚的に表示
  • タスクタイマー：経過時間を表示
  • 作業者管理：複数作業者の同時管理
  • タスク管理：開始、一時停止、完了の管理
  • 完了通知：すべてのタスク完了時に通知
